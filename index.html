Dashboard de Vendas – Equipamentos Eletrônicos

Projeto fictício de dashboard voltado para análise de vendas de equipamentos eletrônicos. Inclui filtros por produto e período, com dados simulados entre os anos de 2015 a 2025 para demonstrar maturidade na organização e visualização de dados. Desenvolvido com foco em clareza, responsividade e profissionalismo, utilizando HTML, CSS, JavaScript e Python.
